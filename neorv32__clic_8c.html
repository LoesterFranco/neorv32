◆ neorv32_clic_handler_install()

int neorv32_clic_handler_install ( uint8_t  channel,
void(*)(void)  handler 
)

Install handler functions for core-local interrupt controller (CLIC) interrupts. The CLIC control register bits are listed in NEORV32_CLIC_CT_enum.

Note
The CLIC interrupt uses the NEORV32 runtime environment.
This function automatically installs the CLIC using neorv32_rte_exception_install(uint8_t exc_id, void (*handler)(void)) and activates the CLIC.
The interrupt enable flag of the according device has to be set manually by configuring the actual device,
e.g. neorv32_spi_setup(uint8_t prsc, uint8_t clk_polarity, uint8_t dir, uint8_t data_size, uint8_t irq_en).
Parameters
[in]channelCLIC interrupt channel. See NEORV32_CLIC_CHANNELS_enum.
[in]handlerThe actual handler function for the specified interrupt (function must be of type "void function(void);").
Returns
0 if success, 1 if error (invalid channel)
