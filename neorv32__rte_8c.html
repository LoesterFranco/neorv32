◆ neorv32_rte_enable_debug_mode()

void neorv32_rte_enable_debug_mode ( void  )

Setup NEORV32 runtime environment in debug mode.

Note
This function installs a debug handler for ALL exception and interrupt sources, which gives detailed information about the exception/interrupt. Call this function before you install custom handler functions via neorv32_rte_exception_install(uint8_t exc_id, void (*handler)(void)), since this function will override all installed exception handlers.
Warning
This function should be used for debugging only, since it only shows the uninitialize exception/interrupt, but does not resolve the cause. Hence, it cannot guarantee to resume normal application execution after showing the debug messages.

int neorv32_rte_exception_install ( uint8_t  exc_id,
void(*)(void)  handler 
)

Install exception handler function to NEORV32 runtime environment.

Note
This function automatically activates the according CSR.mie bits when installing handlers for the MTIME (MTI), CLIC (MEI) or machine software interrupt (MSI). The global interrupt enable bit mstatus.mie has to be set by the user via neorv32_cpu_eint(void).
Parameters
[in]exc_idIdentifier (type) of the targeted exception. See NEORV32_EXCEPTION_IDS_enum.
[in]handlerThe actual handler function for the specified exception (function must be of type "void function(void);"). return 0 if success, 1 if error (invalid exc_id or targeted exception not supported).

◆ neorv32_rte_exception_uninstall()

int neorv32_rte_exception_uninstall ( uint8_t  exc_id)

Uninstall exception handler function from NEORV32 runtime environment, which was previously installed via neorv32_rte_exception_install(uint8_t exc_id, void (*handler)(void)).

Note
This function automatically clears the according CSR.mie bits when uninstalling handlers for the MTIME (MTI), CLIC (MEI) or machine software interrupt (MSI). The global interrupt enable bit mstatus.mie has to be cleared by the user via neorv32_cpu_dint(void).
Parameters
[in]exc_idIdentifier (type) of the targeted exception. See NEORV32_EXCEPTION_IDS_enum. return 0 if success, 1 if error (invalid exc_id or targeted exception not supported).
